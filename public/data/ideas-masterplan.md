- 연령·성별·국적 차별 (줌마팬 혐오 등)
→ 아스테리아 해결: 상향식 거버넌스, 투표로 리더 선출, 리더 소환제, 데이터 기반 등급

### 섹션 2: 경제적 편취 구조
- 팬사인회 당첨 = 앨범 구매 수량 기준
- 랜덤 포토카드로 중복 구매 심리 자극
- 멤버별 메시지 개별 구독 → 월 수만 원 강제
- 무료 콘텐츠 단계적 유료화 (V라이브 폐지 등)
- "소비 = 팬 자격"이라는 문화 내면화
→ 아스테리아 해결: 활동 데이터로만 등급 결정, 돈 없어도 최상위 리그 가능, 7:3 원칙 (활동70%:구매30%)

### 섹션 3: 정보 독점 & 불평등
- 스케줄·티켓 정보 운영진 선점 및 공유 통제
- 유료 멤버십 가입자에게만 사전 정보 공개
- 팬카페 등급별 게시판 접근 차단
- 해외팬 언어 장벽으로 정보 접근 이중 차별
- 정보 독점 이용한 되팔기·리셀 이익 편취
→ 아스테리아 해결: 실시간 대시보드 전체 공개, 정보의 완전 민주화, 글로벌 번역 시스템

### 섹션 4: 기획사의 팬 관리 실태
- 위버스·버블 독점 — 팬 이탈 시 모든 접근 차단
- 팬 의견 수렴 창구 없음 — 일방적 정책 통보
- 공식 팬카페 운영진 = 기획사 대리인 역할
- 유사연애 마케팅으로 과몰입·집착 팬덤 조장
- 팬 데이터 수집·활용 과정의 불투명성
→ 아스테리아 해결: 팬이 주인인 자치 플랫폼, 기획사 협조 불필요, 팬이 모이면 기획사가 찾아오는 구조

### 핵심 악순환 (반드시 끊어야 할 것)
기획사의 팬 관리 방식 → 팬카페 권력 구조 양산 → 정보 독점 정당화 → 경제적 편취 강화 → 다시 기획사 이익

### 아스테리아의 답
- 수직적 권력 구조(기획사→운영진→팬덤→개인팬)를 완전히 뒤집어
- 상향식 구조(개인팬→소모임→팬클럽→리그)로 전환
- 모든 권력은 투명한 활동 데이터에서만 나옴
- 오라클 감시자가 부정행위를 자동 감지하되, 감시 자체도 투명하게 공개 (또 다른 권력 남용 방지)

### 의견→투표→결정 프로세스 (상향식 거버넌스):
- 1단계 의견제안: 누구나 자유롭게 의견 올림. 주제 예시: 생파 장소, 총공 일정, 이벤트 기획 등

- 3단계 공식투표: 후보 2~3개로 1인1표 투표. 2일간 진행. 최다 득표안 확정
- 4단계 공지전달: 확정된 결정이 해당 모임 전체에 공지 자동 전달. 상위 모임에도 보고

### 공지사항 시스템:
- 각 소모임별 공지 게시판
- 투표로 확정된 사항이 자동 공지로 등록
- 상위 모임 공지는 하위 모임에 자동 전파
- 긴급 공지는 푸시 알림으로 전달

### 투표 참여 보상:
- 투표 참여: ACT +2
- 의견 제안: SOC +3
- 채택된 의견 제안자: CRE +5 + 스타더스트 100

### 긴급 안건:
- 긴급 사안은 투표기간 24시간 단축
- 긴급 알림 전체 발송

### 의결 기록 영구 보존:

- 1등 팬클럽이 항상 맵 중앙에 위치
- 1등 팬클럽은 크고 밝은 광채 효과 (골드빛 방사형 glow + 파티클)
- 나머지 팬클럽이 1등을 에워싸며 순위순으로 배치 (나선형)
- 팬클럽 크기가 점수에 비례 (높은팬클럽=크고밝게, 낮은팬클럽=작고어둡게)

- 팬클럽 클릭→상세정보+팬덤페이지 이동

### 승격/강등 비교:
- 상단: 상위리그 꼴등 vs 현재리그 1등 점수 비교 바
- 하단: 현재리그 꼴등 vs 하위리그 1등 점수 비교 바
- 점수 가까우면 빨간경고, 여유있으면 초록안전
- 승격존(맵상단 골드라인), 강등존(맵하단 빨간라인) 시각 표시

### 실시간 효과:
- 점수 변동시 팬클럽 아이콘이 맵에서 위아래로 이동
- "우리 팬클럽이 3칸 올라갔다!" 실감

### 미러 랭킹:
- 같은 리그 내 팬클럽끼리 실시간 순위 비교
- 우리 vs 라이벌 직접 비교 카드
- "이번 주 3위 상승!" 변동 알림

- 역전된 만큼 전부 교체 (승격/강등 수 제한 없음)
- 역전 없으면 변동 0 (아무도 안 오르고 안 내려감)
- 리그 건너뛰기 가능 (더스트→플래닛 직행 등, 2단계 이상 역전시)
- 더스트는 강등 없음 (최하위)

#### 절대 점수제 + Hard Gate
- 더스트→스타: S≥800 + 인원10만+ + 포인트5000+ + 게더링250+ + 활동밀도40%+
- 스타→플래닛: S≥850 + 인원50만+ + 테리토리50+ + 베이스1000+ + 3개월우상향 + 정화1000건+
- 플래닛→노바: S≥900 + 인원500만+ + 거버넌스리더80%합의
- 노바→퀘이사: S≥950 + 인원1000만+ + 글로벌방어체계 + 최고도덕기준

#### 매일 노출 시스템 (모든 리그, 모든 팬에게 상시)
- 리그 메인: 상위 꼴등 vs 현 리그 1위 점수 비교 / 현 리그 꼴등 vs 하위 1위 점수 비교
- 팬덤 메인: 우리 팬클럽 관련 승격/강등 전선
- 모든 팬 화면: 상시 위젯 (순위/점수/격차/D-데이)
- 5개 리그 전체 구조:
  - 퀘이사: 하단만 (퀘이사꼴등 vs 노바1위)
  - 노바: 상단(퀘이사꼴등 vs 노바1위) + 하단(노바꼴등 vs 플래닛1위)
  - 플래닛: 상단(노바꼴등 vs 플래닛1위) + 하단(플래닛꼴등 vs 스타1위)
  - 스타: 상단(플래닛꼴등 vs 스타1위) + 하단(스타꼴등 vs 더스트1위)
  - 더스트: 상단만 (스타꼴등 vs 더스트1위)

#### 격차 시각 효과
- 100점이상: 🟢여유(초록) / 50~99: 🟡주의(노란) / 20~49: 🟠긴박(주황) / 20미만: 🔴임박(빨강펄스) / 역전: 💥골드플래시+전체알림

#### 경쟁 자극 요소
- 역전 임박 경보: 50점 이내시 양쪽 경보
- 역전 히스토리: 시즌 내 모든 역전 순간 타임라인 기록
- 승격 배틀존: 기준점 근처 팬클럽 특별 표시
- 승격 플레이오프: 근소차시 72시간 최종 결전
- 강등 방어전: 역전당하면 72시간 방어 미션 기회
- 리그 건너뛰기시: "워프 점프" 전설 업적 + 특수 세레모니
- 재기 보너스: 강등 팀 다음 시즌 AP 1.5배 (2주)
- 기적의 역전: 강등존→승격시 특수 업적
- 시즌 종료: 최종 점수 확정 → 워프(승격)/블랙홀(강등) 동시 연출

### Q3 확정: 강등 보호 장치 → "특혜 없음, 경고+화력 집중 유도"

#### 핵심 원칙
- 아스테리아에 특혜는 없다. 시스템은 위험을 알려주고, 싸울 환경을 만들어줄 뿐

#### 경고 시스템 (단계별 확대)
- D-21: 🟡 1차 경보 "위기의 성좌" (강등위험 팬클럽 공개, 점수격차 공개)
- D-14: 🟠 2차 경보 "임박한 블랙홀" (푸시알림, IAI/GSI/PII 엑스레이, 강등확률 예측)

- D-3: 🔥 최후의 3일 돌입
- 리그별 경고 시작: 스타 D-14 / 플래닛·노바 D-21 / 퀘이사 D-30

#### 최후의 3일 — 화력 집중 시스템 8가지
- AP 배율: 평소와 동일 (1배). 특혜 없음
1. 실시간 전광판: 1분마다 점수 비교, 격차 줄어들면 화면 테두리 빨갛게
2. 화력 미터: 팬클럽 활동량 게이지, 시간대별 히트맵, 100% 넘으면 오버차지
3. 긴급 소집 알림: 접속자 수 실시간, 리더가 집중시간 예약(상향식 제안)
4. 연쇄 미션 폭탄: 미션 2배 증가, 완료→다음 즉시 해금
5. 전선 지도: 승격↑강등↓ 화살표, 역전시 전서버 알림, 역전 히스토리
6. 소모임 기여도 배틀: 소모임 순위 공개, TOP3 "최후의 수호자" 뱃지(영구)+스타더스트
7. 실황 중계 메시지: 시스템 자동 상황 중계
8. 결전 카운트다운: 시분초→마지막1시간 초단위+심장박동→0초 점수확정

#### 점수 확정

## 34. 소원의 성궤 + 국민 투표 시스템 ✅ (#57)

### 소원의 성궤 (Wish Orbit) — 집단 소원 달성 시스템

#### 핵심 컨셉
팬클럽 전체가 하나의 소원을 정해 에너지를 모아 달성하는 집단 목표 시스템.
상향식 파이프라인으로 소원이 소모임→상위→팬클럽까지 올라오는 구조.

#### 소원 제안
- 자격: 전원 가능 (완전 민주적)
- 제한: 1인 월 1개만 제안 가능 (장난/어뷰징 방지)
- 제안 장소: 본인 소속 소모임 내부에 제안

#### 상향식 소원 채택 파이프라인 (투표 없음, 공감만)
- 개인이 소속 소모임에 소원 제안
- 소모임 내부에서 공감(좋아요) 모집 (7일간)
- 해당 모임 멤버 30% 이상 공감 시 상위 모임으로 자동 전달
- 상위 모임에서 다시 공감 모집 → 30% 이상 시 더 상위로
- 팬클럽 전체까지 올라온 소원들 중 공감 상위 → 자동 확정
- 7일간 30% 미달 시 자동 소멸

#### 리그별 파이프라인 경로
- 더스트: 개인 → 포인트(200명) → 게더링(2천) → 팬클럽
- 스타: 개인 → 유닛(200명) → 베이스(2천) → 테리토리(2만) → 팬클럽
- 플래닛: 개인 → 유닛(250명) → 베이스(2500) → 테리토리(2.5만) → 팬클럽
- 노바: 개인 → 라운지(100명) → 스퀘어(1천) → 디스트릭트(1만) → 프로빈스(10만) → 팬클럽
- 퀘이사: 개인 → 오빗(100명) → 클러스터(1천) → 섹터(8천) → 도미니언(4만) → 엠파이어(20만) → 팬클럽

#### 소원 구조: 3+1
- 메인 소원 1개: 팬클럽 전체 화력 집중 (공감 1위)
- 서브 소원 2개: 소규모 달성 가능 (공감 2~3위)
- 서프라이즈 소원 1개: 시즌 시작 시 시스템 자동 제안 (도전과제 형태)

#### 소원 카테고리 6종
1. 이벤트 소원: 온라인 생파, 팬미팅 준비 등
2. 콘텐츠 소원: 팬아트 대회, 아티스트 역사 영상 제작 등
3. 플랫폼 소원: 새 방 아이템, 전용 이모지 등
4. 커뮤니티 소원: 타 팬덤 교류전, 멘토링 등
5. 기부/사회공헌 소원: 아티스트 이름 기부, 숲 조성 등
6. 도전 소원: 리그 승격, 서버 TOP 3 진입 등

#### 에너지 모으기
- 방식: 활동 연동 자동 기부 + 소원 전용 미션
- 일반 활동 시 자동 소량 축적 (출석 +10, 미션 +30 등)- 소원 전용 미션 리스트 → 완료 시 대량 에너지 축적
- 스타더스트 기부는 보조 수단
#### 에너지 시각화: 별 키우기
- 소원이 작은 별로 시작
- 에너지 모일수록 별이 커지고 밝아짐
- 단계: 먼지→작은별→중간별→큰별→거대별→초신성
- 목표 달성 시 초신성 폭발 연출!
- 리그 테마색 적용 (더스트 회색 / 스타 금색 / 플래닛 에메랄드 / 노바 보라 / 퀘이사 골드)

#### 달성 시
- 전체 서버 축하 이벤트 (30분간 특수 이펙트)
- "소원의 별" 팬클럽 하늘에 영구 생성 (해당 소원 이름 새겨짐)
- 기여도별 뱃지: 상위10% 골드 / 상위30% 실버 / 나머지 브론즈
- 소원 보물상자: 기여자에게 랜덤 아이템 드롭 (기여도 높을수록 좋은 확률)
- 소원 아카이브 영구 기록 ("우리가 이뤘던 소원들" 역사관)

#### 미달성 시: 달성률 차등 환불
- 80%+ 달성: 에너지 90% 환불 + 부분 보상
- 50~79% 달성: 에너지 70% 환불
- 50% 미만: 에너지 50% 환불
- "미완의 별" 기록 (노력한 흔적 보존)
- 다음 시즌 이월 가능

#### 리그별 에너지 목표치
- 더스트 (2만명): 50,000
- 스타 (10만명): 200,000
- 플래닛 (50만명): 800,000
- 노바 (500만명): 5,000,000
- 퀘이사 (1000만명): 15,000,000

#### 리그 통합/분리
- 분리: 소원 제안 경로, 에너지 목표치, 소원 종류/규모, 보상 차등
- 통합: 시각화(별 키우기), 아카이브, 기본 시스템 로직

### 국민 투표 연출 (Sovereign Vote Ceremony)

#### 핵심 원칙: 투표 최소화 (피로감 방지)
- 투표는 정말 중요한 3가지만:
  1. 리더 선출/소환 (상향식 거버넌스 핵심)
  2. 마을 변경 (팬클럽 전체 영향)
  3. 규칙 변경 (팬클럽 전체 영향)
- 소원 채택, 이벤트 결정, 사면 등은 투표 없이 처리

#### 4단계 프로세스
1. 발의 (Day 0): 안건 등록 → 별빛 신호탄 이펙트 → 전체 알림
2. 토론 (72시간): 찬반 의견 게시판, 실시간 여론 그래프 (찬성=금빛, 반대=은빛)
3. 투표 (24시간): 투표 개시 세레모니 → 별빛 투사 이펙트 → 실시간 은하수 현황판
4. 결과 발표: 카운트다운 → 결과 공개 → 세레모니 연출

#### 투표 연출
- 투표 시 아바타가 별을 하늘로 던지는 애니메이션
- 찬성=금빛 별, 반대=은빛 별
- 모든 투표자의 별이 팬클럽 하늘에 실시간 축적 → 금빛 은하수 vs 은빛 은하수
- 투표 마감 직전 1시간: "최후의 별빛" 모드 (배경 어두워지고 별만 반짝)

#### 결과 발표 연출
- 찬성 승리: 금빛 폭죽 + "팬클럽의 뜻이 모였습니다! ✨" + 30초간 금빛 은하수 배경
- 반대 승리: 은빛 안개 + "팬클럽이 신중한 선택을 했습니다 🌙" (부정적 표현 안 씀)
- 박빙 (55:45 이내): "역사적인 접전!" + 투표 참여자 전원 "결정의 증인" 뱃지

#### 투표 보상
- 투표 참여: ACT+2, LOY+1
- 토론 참여: SOC+3
- 의견 제안자(채택된 안건): CRE+5, 스타더스트 100
- 시즌 최다 투표 참여: "민주주의의 별" 칭호

#### 리그 통합/분리
- 통합: 투표 프로세스(4단계), 연출/이펙트(리그 테마색만 차등), 보상, 긴급 투표(24시간 단축)
- 분리: 의결 범위/권한(masterplan 기존 확정 기준), 토론 기간(퀘이사는 인원 많아 더 길게 가능)
