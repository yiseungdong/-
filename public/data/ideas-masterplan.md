## 11. 팬덤 문제 현황 분석 (사업계획서 기반, 2026.03)

아스테리아는 아래 4대 문제를 해결하기 위해 만들어진 플랫폼이다.
모든 기능 기획 시 이 문제들을 해결하는 방향인지 반드시 확인할 것.

### 섹션 1: 권력 남용 & 내분·갈등
- 팬카페 운영진 단독 임명, 선출 기준 없음
- 게시물 삭제·강퇴 권한 남용, 이의제기 창구 부재
- 친목 운영진 비위 묵인, 일반 회원 역차별
- 운영 방침 공지 없이 규정 자의적 적용
- 고참/신참 서열 문화, 소비 금액 기준 팬 등급 서열화
- 특정 멤버 비선호 팬 집단 배척·왕따
- 연령·성별·국적 차별 (줌마팬 혐오 등)
→ 아스테리아 해결: 상향식 거버넌스, 투표로 리더 선출, 리더 소환제, 데이터 기반 등급

### 섹션 2: 경제적 편취 구조
- 팬사인회 당첨 = 앨범 구매 수량 기준
- 랜덤 포토카드로 중복 구매 심리 자극
- 멤버별 메시지 개별 구독 → 월 수만 원 강제
- 무료 콘텐츠 단계적 유료화 (V라이브 폐지 등)
- "소비 = 팬 자격"이라는 문화 내면화
→ 아스테리아 해결: 활동 데이터로만 등급 결정, 돈 없어도 최상위 리그 가능, 7:3 원칙 (활동70%:구매30%)

### 섹션 3: 정보 독점 & 불평등
- 스케줄·티켓 정보 운영진 선점 및 공유 통제
- 유료 멤버십 가입자에게만 사전 정보 공개
- 팬카페 등급별 게시판 접근 차단
- 해외팬 언어 장벽으로 정보 접근 이중 차별
- 정보 독점 이용한 되팔기·리셀 이익 편취
→ 아스테리아 해결: 실시간 대시보드 전체 공개, 정보의 완전 민주화, 글로벌 번역 시스템

### 섹션 4: 기획사의 팬 관리 실태
- 위버스·버블 독점 — 팬 이탈 시 모든 접근 차단
- 팬 의견 수렴 창구 없음 — 일방적 정책 통보
- 공식 팬카페 운영진 = 기획사 대리인 역할
- 유사연애 마케팅으로 과몰입·집착 팬덤 조장
- 팬 데이터 수집·활용 과정의 불투명성
→ 아스테리아 해결: 팬이 주인인 자치 플랫폼, 기획사 협조 불필요, 팬이 모이면 기획사가 찾아오는 구조

### 핵심 악순환 (반드시 끊어야 할 것)
기획사의 팬 관리 방식 → 팬카페 권력 구조 양산 → 정보 독점 정당화 → 경제적 편취 강화 → 다시 기획사 이익

### 아스테리아의 답
- 수직적 권력 구조(기획사→운영진→팬덤→개인팬)를 완전히 뒤집어
- 상향식 구조(개인팬→소모임→팬클럽→리그)로 전환
- 모든 권력은 투명한 활동 데이터에서만 나옴
- 오라클 감시자가 부정행위를 자동 감지하되, 감시 자체도 투명하게 공개 (또 다른 권력 남용 방지)

## 16. 리그 모임 커뮤니케이션 시스템

### 모임별 대화방:
- 각 리그 소모임(오빗/유닛/포인트 등)마다 전용 대화방
- 별자리 친구 그룹 채팅방도 별도 존재
- 상위 모임(클러스터/베이스/게더링)도 대화방 있음

### 의견→투표→결정 프로세스 (상향식 거버넌스):
- 1단계 의견제안: 누구나 자유롭게 의견 올림. 주제 예시: 생파 장소, 총공 일정, 이벤트 기획 등
- 2단계 공감투표: 올라온 의견에 공감(좋아요) 누름. 3일간 진행. 공감 상위 2~3개가 자동으로 후보 선정
- 3단계 공식투표: 후보 2~3개로 1인1표 투표. 2일간 진행. 최다 득표안 확정
- 4단계 공지전달: 확정된 결정이 해당 모임 전체에 공지 자동 전달. 상위 모임에도 보고

### 공지사항 시스템:
- 각 소모임별 공지 게시판
- 투표로 확정된 사항이 자동 공지로 등록
- 상위 모임 공지는 하위 모임에 자동 전파
- 긴급 공지는 푸시 알림으로 전달

### 투표 참여 보상:
- 투표 참여: ACT +2
- 의견 제안: SOC +3
- 채택된 의견 제안자: CRE +5 + 스타더스트 100

### 긴급 안건:
- 긴급 사안은 투표기간 24시간 단축
- 긴급 알림 전체 발송

### 의결 기록 영구 보존:
- 모든 투표 결과 영구 기록 ("2026.05 생파장소: 서울OO홀 찬성73%")

## 17. 리그 랭킹 맵 (league.html) — Phase 2 #19

### 구성: 5개 화면 (리그별 1개씩)
- 왼쪽 네비에서 리그 전환 (더스트/스타/플래닛/노바/퀘이사)
- 각 리그별 모든 팬클럽이 맵에 표시

### 맵 디자인:
- 맵이 화면의 3~4배로 넓게 펼쳐짐
- 마우스 드래그/모바일 터치로 맵 이동하며 탐험
- 1등 팬클럽이 항상 맵 중앙에 위치
- 1등 팬클럽은 크고 밝은 광채 효과 (골드빛 방사형 glow + 파티클)
- 나머지 팬클럽이 1등을 에워싸며 순위순으로 배치 (나선형)
- 팬클럽 크기가 점수에 비례 (높은팬클럽=크고밝게, 낮은팬클럽=작고어둡게)
- 내 팬클럽은 골드 하이라이트+깜빡임으로 눈에 띄게
- 팬클럽 클릭→상세정보+팬덤페이지 이동

### 승격/강등 비교:
- 상단: 상위리그 꼴등 vs 현재리그 1등 점수 비교 바
- 하단: 현재리그 꼴등 vs 하위리그 1등 점수 비교 바
- 점수 가까우면 빨간경고, 여유있으면 초록안전
- 승격존(맵상단 골드라인), 강등존(맵하단 빨간라인) 시각 표시

### 실시간 효과:
- 점수 변동시 팬클럽 아이콘이 맵에서 위아래로 이동
- "우리 팬클럽이 3칸 올라갔다!" 실감

### 미러 랭킹:
- 같은 리그 내 팬클럽끼리 실시간 순위 비교
- 우리 vs 라이벌 직접 비교 카드
- "이번 주 3위 상승!" 변동 알림

#### 승격 세레모니 "워프 점프"
- 카운트다운 종료→화면 정지→결과 발표(골드 빛줄기)→워프 애니메이션→축하 폭탄
- 리그별 차등: 더스트→스타(은색) / 스타→플래닛(에메랄드 오로라) / 플래닛→노바(초신성) / 노바→퀘이사(전서버 이벤트)
- 건너뛰기: "더블/트리플 워프" 추가 연출

#### 팬클럽 전체 변화
- 조직 구조 자동 전환 (하위→상위 리그 체계), 리더 투표 선출
- 팬덤 마을 업그레이드, UI 테마 변경
- 리그 전용 기능 해금: 스타(유닛협동미션,거래소판매) / 플래닛(전략회의,방어미션,글로벌연결) / 노바(대외미션,분업,합동스트라이크) / 퀘이사(기획사응답,연대기,소버린게이트최고권한)

#### 개인 보상
- 승격 기여 뱃지(영구): 골드(상위10%)/실버(상위30%)/브론즈(나머지) — 기여도 AP총합 기준
- 스타더스트: 더스트→스타 500 / 스타→플래닛 1000 / 플래닛→노바 2000 / 노바→퀘이사 5000 (상위10% +추가, 건너뛰기 2배)
- 캐릭터 이름 변경 1회 무료
- 스탯 보너스: 가장 많이 활동한 스탯에 핵심+10/주요+5/참여+2
- 방 해금 아이템(돈으로못삼): 스타(별빛벽지,은하수바닥) / 플래닛(행성모빌,우주창문) / 노바(오로라천장,보라포탈) / 퀘이사(홀로그램벽,왕좌,우주전경)
- 승격 기념 별똥별 (리그색, 추억카드)
- 강등해도 해금 아이템 영구 소유

#### 소모임 보상
- MVP 소모임 TOP3: 1등 "승격의 주역"+스타더스트1000 / 2등 "핵심전력"+500 / 3등 "든든한기둥"+300 (1시즌 한정 칭호)

#### 특수 케이스
- 건너뛰기 승격: "전설의 도약" 업적 + 모든 보상 2배
- 시즌 1위 승격: "시즌 챔피언" 업적 + 왕관마크 1시즌 + 챔피언 오라 1시즌
- 퀘이사 승격: 전서버 30분 골드은하수 배경 + 전체 기념아이템 + 24시간 메인화면 표시

### Q5 확정: 강등시 변화/처리

#### 강등 세레모니 "성운 회귀"
- 부드러운 블루 빛 감싸며 "성운으로 돌아갑니다" (부정적 단어 사용 안함)
- 은빛 먼지 흩뿌리며 부드럽게 하강 (씨앗이 내려앉는 느낌)
- 위로 메시지: "별빛은 사라지지 않습니다. 보물창고에 영원히 보관됩니다."
- 리그별 차등: 스타→더스트(은빛) / 플래닛→스타(초록→금) / 노바→플래닛(보라→에메랄드) / 퀘이사→노바(골드→보라+전서버)
- 건너뛰기: "대성운 회귀"

#### 팬클럽 변화
- 조직 구조 자동 축소 전환, 리더 재투표
- 마을 축소 (기존 건물 "잔해" 남음→추억 흔적, 클릭시 추억카드)
- 리그 전용 기능 잠금 (아이콘은 보임+자물쇠, 재승격 욕구 자극)
- UI 테마 변경

#### 개인 변화
- 절대 안 사라지는 것: 모든 뱃지/아이템/스타더스트/레벨/외형/별똥별/레어카드/히스토리/해금아이템
- 변경되는 것: 리그표시/리그랭킹/사용가능기능/조직소속
- "前 리그명" 칭호 사용 가능 (표시 선택은 개인), 강등 기념 블루 별똥별
- "재기의 불꽃" 마크 → 재승격시 "기적의 귀환" 업적, 2시즌 연속 "불사조" 업적

#### 소모임 변화
- 해체가 아닌 "재편", 기존 기록 보존, 같은 이름 재결성 가능
- 별자리 친구 관계 유지 (인원수만 하위리그 기준 조정)

#### 재기 시스템
- "성운 재건 로드맵" 자동 가이드 (부족한 지표 분석+목표 제시)
- 적응 미션 (승격 팀과 동일 구조, 특혜 아님)
- 재기 업적: 1시즌 재승격 "기적의 귀환" / 2시즌 연속 "불사조" / 건너뛰기 재승격 "초신성 폭발"
- 건너뛰기 강등: 목적지 리그 구조로 바로 전환

### Q6 확정: 시즌 리셋 범위 → "리셋 없음, 전체 누적"

#### 핵심 원칙
- 아스테리아에서는 아무것도 사라지지 않는다. 모든 활동은 영원히 누적된다
- 팬덤은 살아있는 생물처럼 스스로 오르내린다 (아티스트 활동기/공백기에 따라 자연 변동)

#### 리셋되는 것: 리더 직책만
- 모든 소모임 리더 매 시즌 재투표 (상향식, 재선 가능, 권력남용 방지)

#### 영구 유지: 나머지 전부
- AP, 6대스탯, S점수, IAI/GSI/PII, 캐릭터, 아이템, 스타더스트, 뱃지, 별똥별, 친구, 팬클럽소속, 성궤번호, 히스토리, 거래기록, 투표기록

#### 승격/강등 판정: 전체 누적 점수
- 시스템 리셋 없이 팬덤 자연 생태계에 의존
- 활동기 팬클럽 → 점수 급상승, 공백기 팬클럽 → 성장 둔화 → 역전 가능

#### 시즌 전환 시퀀스
- 시즌 종료(지정 날짜+시간) → 점수확정 → 승격/강등 발표+세레모니
- 휴식기 1주: 리더 재투표, 조직 재편, 결산 리포트, 적응 미션 (활동 가능, 점수도 누적)
- 새 시즌 개막: 세레모니 + 본격 경쟁

#### 시즌 결산 리포트 (자동생성, 보물창고 영구 보관)
- 개인: 시즌AP/스탯변동/순위/아키타입/활동TOP3/기여도/한줄요약
- 팬클럽: 최종순위/결과/에너지/MVP소모임TOP3/MVP개인TOP10/극적순간

#### 시즌 표기: 번호+고유이름 (예: "시즌1: 개척의 시대")
