# 아스테리아 실사용 테스트 체크리스트

## 테스트 환경
- [ ] PC 크롬 (데스크탑)
- [ ] 모바일 크롬 (안드로이드)
- [ ] 모바일 사파리 (아이폰) — 가능한 경우

## 페이지별 점검

### intro.html (인트로)
- [ ] 별 애니메이션 정상 표시
- [ ] 성궤번호 표시
- [ ] 입장 버튼 클릭 → index.html 이동

### index.html (홈/대시보드)
- [ ] 별빛 배경 정상 표시
- [ ] 상단 네비게이션 6개 메뉴 표시
- [ ] 리그 맵 표시
- [ ] 모바일에서 터치/스크롤 정상

### astra.html (아스트라)
- [ ] 아바타 표시
- [ ] 레벨/스탯 정보 표시
- [ ] 모바일 레이아웃 정상

### room3d.html (내 방)
- [ ] 3D 방 로딩
- [ ] 캐릭터 표시
- [ ] 클릭으로 이동
- [ ] 마우스/터치 드래그로 회전
- [ ] 모바일에서 터치 정상

### chat.html (채팅)
- [ ] AI 아티스트 대화 가능
- [ ] 감정 이모지 버튼 작동
- [ ] 대화 시간 카운트다운 표시
- [ ] 별빛 다이어리 열기/닫기

### league.html (리그)
- [ ] 5개 리그 전환 가능
- [ ] 팬클럽 맵 표시
- [ ] 드래그 탐험 가능
- [ ] 승격/강등 비교 바 표시

### stats.html (스탯)
- [ ] 헥사곤 레이더 차트 표시
- [ ] 6스탯 수치 표시
- [ ] 상위 X% 표시

### archetype.html (아키타입)
- [ ] 15종 아키타입 카드 표시
- [ ] SNS 공유 버튼 작동

### shop.html (상점)
- [ ] 6개 카테고리 탭 전환
- [ ] 아이템 목록 표시
- [ ] NPC 상인 대화
- [ ] 뽑기 시스템 작동
- [ ] 구매 → 스타더스트 차감

### nebula.html (성궤)
- [ ] 개인 공간 표시
- [ ] 아티스트 캐릭터 초대/대화

### avatar.html (아바타 꾸미기)
- [ ] 7개 탭 표시
- [ ] 아이템 적용/미리보기
- [ ] 프리셋 저장/불러오기

### admin.html (관리자)
- [ ] 7개 탭 전환
- [ ] 대시보드 KPI 표시
- [ ] 관리자 전용 접근 (일반 유저 접근 차단 확인)

## 공통 점검
- [ ] 모든 페이지 별빛 배경 정상
- [ ] 모든 페이지 네비게이션 정상 이동
- [ ] 모든 페이지 모바일 반응형
- [ ] 토스트 메시지 정상 표시
- [ ] 로딩 시 3초 이내 표시
- [ ] 콘솔에 심각한 에러 없음 (F12 → Console 확인)

## 승격/에너지 점검
- [ ] 출석 → AP 증가 확인
- [ ] 활동 → 스탯 반영 확인
- [ ] 스타더스트 획득/소모 정상
- [ ] 리그 순위 표시 정상
